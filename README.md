## Overview
This toolkit is built on the principles of **Polyvagal Theory** and **Restorative Psychology**. Unlike standard stress management tools which focus on optimization, this toolkit focuses on safety and nervous system regulation.

## Included Tools
1. **The Rescue Canvas:** A one-page dashboard for immediate triage and boundary setting.
2. **The Full Workbook:** A deep-dive guide for understanding the "Dorsal Vagal" shutdown and creating a long-term recovery plan.
